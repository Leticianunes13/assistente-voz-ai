# 🎙️ Assistente de Voz & Acessibilidade Inteligente

Este projeto nasceu como parte dos meus estudos de tecnologia, impulsionado pelo **Bootcamp Santander / Bradesco**, com o objetivo de unir inteligência artificial e acessibilidade. 

O sistema processa comandos de voz para gerar respostas inteligentes em tempo real.

## 🛠️ Tecnologias Atuais
- **Whisper (OpenAI):** Transcrição de alta precisão (Speech-to-Text).
- **Groq Cloud (LLaMA 3.1):** Processamento de linguagem natural ultra-rápido.
- **gTTS:** Síntese de voz para retorno em áudio (Text-to-Speech).
- **Python & Google Colab:** Ambiente inicial de desenvolvimento.

## 🚀 Próximos Passos (Upgrades Futuros)
Atualmente, estou trabalhando na transição deste projeto para o **VS Code**, visando as seguintes melhorias:

* **🎬 Resumo de Vídeos do YouTube:** Integração para analisar e resumir conteúdos diretamente via link.
* **📄 Análise de Páginas Web:** Extração de contexto e resumo de sites apenas informando a URL.
* **🌍 Multilinguismo Total:** Implementação de tradução simultânea, permitindo que o idioma de saída seja diferente do idioma de entrada.
* **🖥️ Interface Local:** Migração para uma aplicação desktop ou web completa, saindo do ambiente de notebooks.

## 📂 Como testar
1. Configure sua `GROQ_API_KEY` nos segredos do ambiente.
2. Execute o notebook e grave seu áudio.
